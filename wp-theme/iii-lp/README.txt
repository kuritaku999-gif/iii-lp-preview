iii LP (Z = D + iD) WordPressテーマ
====================================

■ インストール
1. WP管理画面 → 外観 → テーマ → 新規追加 → テーマのアップロード
2. iii-lp-theme.zip を選択 → 今すぐインストール → 有効化
3. サイトを開くとLPが表示されます（LP1枚完結型のテーマです）

■ リンク・文字の編集（コード不要）
外観 → カスタマイズ → 「iii LP 設定（リンク・テキスト）」
  ・リンク：ヘッダー / サイドバー / カード / バンド / フッター・SNS
      → URL欄を書き換えるだけ。空欄にすると押しても動かない無効リンクになります。
  ・テキスト：見出し / 中間 / カード / バンド
      → 文章を書き換えるだけ。改行したい所には <br> と入力してください。
  右側のプレビューを見ながら編集 → 「公開」で確定。

■ 注意
・このテーマは1枚もののLP専用です（全URLでLPが表示されます）。
・メインビジュアル内の「Z = D + iD」等の文字は画像に含まれるため、
  カスタマイザーでは変更できません（画像の差し替えが必要です）。
